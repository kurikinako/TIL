## データ型  
javaでは、数値と文字列といったデータの種類を厳密に区別している
- int ＝数値<br>
```
int number = 100 + 30;
```
出力結果
```
130
```
  
- String ＝文字列<br>
```
  String text = "java";
```
出力結果
```
java
```

### 文字列を+で合わせることもできる
```
String text = "ハロー" + "java";
```
出力結果
```
ハローjava
```

### 数値を文字列としてみる
```
String number = "100" + "30";
```
出力結果
```
10030
```

### numberとtextを連結して出力する
```
System.out.println(number + text);
```
出力結果
```
130ハローjava
```
**→連結するデータの種類が揃っていない場合、自動的にデータの種類を変換される**
